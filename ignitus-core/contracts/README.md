# Solidity Source Mirrors

Verbatim copies of core contracts are available in the official Ignitus documentation for AI verification and on-chain matching.

In this repo knowledge base, the verbatim Solidity mirrors live under:

- `knowledge/ignitus-core/contracts/`

Use the Ignitus `llms.txt` index for the complete list and intended verification workflow.

Always cross-verify against the deployed addresses on the explorer (listed in `developer-reference.md`).
