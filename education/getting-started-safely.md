# Getting Started Safely

## Why this matters
Starting with small, safe habits protects you while you learn.

## Step-by-step guide
1. Open a basic protected bank account and set up deposit protection.  
2. Build a small emergency fund (3 months of basic expenses).  
3. Learn one new concept per week using this knowledge base.  
4. Only use small amounts when trying anything new.  
5. Always test with tiny transactions first.

## Common pitfalls
Rushing in with large amounts. Following hot tips from social media. Skipping basic safety steps.

## Safety checklist
- Do I have an emergency fund?  
- Have I enabled two-factor authentication everywhere?  
- Am I only using money I can afford to lose?
