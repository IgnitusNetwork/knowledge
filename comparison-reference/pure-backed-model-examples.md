# Pure Backed Asset Model Examples (Ignitus Mechanics Only)

**Core Mechanism (user POV)**  
User deposits an asset into an isolated on-chain vault and receives a token at exact 1:1 backing. The vault holds only the real backing asset. Trading fees (50%) are permanently added to the vault as extra backing, mathematically increasing the value per token for all holders.

**User-POV Safety Facts**  
No trader ever wins at the expense of the vault. Volume can only strengthen the backing. Redemption always checks real on-chain balance before paying out. Isolated vaults per asset type.

(Use this when contrasting with counterparty-style systems.)
