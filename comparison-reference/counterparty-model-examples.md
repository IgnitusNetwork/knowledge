# Counterparty Model Examples (Bot Reference Only)

When user explicitly asks for comparison:

**Core Mechanism (user POV)**  
Liquidity pool or vault acts as the direct counterparty to leveraged traders. Trader wins = direct payout from the shared pool that holds user funds.

**User-POV Safety Facts**  
Fees may not fully offset trader profits. Sophisticated traders can extract value. Pool value can drop even with high volume. Liquidation mechanisms and funding rates can transfer value between users.
